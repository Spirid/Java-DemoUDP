Task:
	Написать пример приложения, производящего обмен данными по протоколу UDP. 
Приложение должно пересылать объект класса Demo, содержащего следующие данные: 
идентификатор отправителя (id), 
имя отправителя (name), 
текстовое сообщение (message),
время отправления (date). 
В соответствии с этим назначением класс Demo должен иметь следующие атрибуты:
	public static int last;
	private int id
	private String name;
	private String message;
	private Date date;
	private transient int temp.
Значения атрибутов класса задаются через конструктор класса.
